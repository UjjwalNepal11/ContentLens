## Features

- **Deep Content Analysis** - Advanced AI algorithms analyze content at multiple levels
- **Advanced Summarization** - Get concise, accurate summaries that capture the essence of any text
- **Sentiment & Tone Detection** - Understand emotional context and tone variations
- **Keyword & Insight Extraction** - Identify key terms and topics
- **Readability & Structure Scoring** - Measure readability scores with improvement suggestions
- **Image Analysis** - OCR-powered text extraction from images using Tesseract.js
- **Saved Analysis History** - All analyses are securely stored and accessible anytime
- **Visual Analytics Dashboard** - Track trends with interactive charts
- **PDF Export** - Export analysis results to PDF format

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Next.js API Routes, Prisma ORM
- **Database**: PostgreSQL
- **Authentication**: Clerk
- **AI**: Google Generative AI
- **State Management**: Zustand
- **Rate Limiting**: Upstash
- **PDF Generation**: pdf-lib
